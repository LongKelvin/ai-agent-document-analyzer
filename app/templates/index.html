How it works

This demo analyzes documents using an AI agent that:

  • Uses RAG (Retrieval-Augmented Generation) with embeddings
  • Treats LLM output as untrusted and validates with Pydantic
  • Provides evidence-based analysis (no hallucination)
  • Returns structured JSON matching a strict schema
